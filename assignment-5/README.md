# Python Modules & Packages Assignment

## Tasks Covered
- Task 1: Create Module - math_utils (add, subtract, square)
- Task 2: Create Module - string_utils (capitalize_words, reverse_string, word_count)
- Task 3: Create Package - shop_package
  - discount.py (apply_discount, flat_discount)
  - billing.py (calculate_total, apply_tax)
  - __init__.py (import functions for direct access)
- Task 4: Importing Modules & Package in main.py
