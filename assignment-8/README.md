# Streamlit Assignment (Basic App Building)

## Description
This assignment demonstrates building simple interactive web apps using Streamlit. It focuses on user input, displaying results, and basic UI components.

## Tasks Covered
- Task 1: Basic App (text input + button)
- Task 2: Price Calculator (number input + slider)
- Task 3: Product Form (sidebar inputs)
- Task 4: Mini Dashboard (selectbox + chart)

## Folder Structure
streamlit_assignment/
- app_basic.py
- app_discount.py
- app_product_form.py
- app_dashboard.py

## How to Run

1. Install Streamlit: pip install streamlit
2. Run any app: streamlit run app_basic.py


## Features
- Simple and clean UI
- Use of Streamlit components:
  - st.title()
  - st.text_input()
  - st.button()
  - st.slider()
  - st.selectbox()
  - st.sidebar
  - st.success()
  - st.table()
  - st.bar_chart()

## Notes
- No advanced features used (as per assignment restrictions)
- No APIs, sessions, or databases used
