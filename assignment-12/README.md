# Seaborn Assignment

## Description
This assignment demonstrates the use of Seaborn for Exploratory Data Analysis (EDA) using a Retail Sales Dataset. It covers relational plots, distribution plots, matrix plots, categorical plots, regression plots, and figure-level plots.

## Dataset Used
Retail Sales Dataset (Kaggle)

## Tasks Covered

- Task 1: Relational Plot
- Task 2: Line Plot as Scatter & Facet
- Task 3: Distribution Plots
- Task 4: Bivariate Distribution Plots
- Task 5: Matrix Plots
- Task 6: Categorical Plots
- Task 7: Regression Plots
- Task 8: Multi-Plots & Figure-Level Plots

## Libraries Used

- Pandas
- Seaborn
- Matplotlib

## How to Run

1. Download the Retail Sales Dataset from Kaggle.
2. Place the CSV file in the project directory.
3. Open `seaborn_assignment.ipynb`.
4. Run all cells sequentially.
