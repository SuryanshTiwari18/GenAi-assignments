# Matplotlib Assignment

## Description
This assignment demonstrates core Matplotlib plotting techniques including line plots, scatter plots, bar plots, multiple bars, stacked bar charts, histograms, and pie charts using a sales dataset.

## Tasks Covered
- Task 1: Line Plot (Sales Trend)
- Task 2: Scatter Plot
- Task 3: Bar Plot (Vertical & Horizontal)
- Task 4: Multiple Bar Plot
- Task 5: Stacked Bar Chart
- Task 6: Histogram
- Task 7: Pie Chart

## Dataset Used
Kaggle Retail Sales Dataset

## How to Run
1. Download the sales dataset from Kaggle.
2. Place the CSV file in the same folder as the notebook.
3. Open the notebook: `matplotlib_assignment.ipynb`
4. Run all cells sequentially.
