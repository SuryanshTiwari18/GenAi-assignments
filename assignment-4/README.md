# Python File Handling Assignment

## Description
This assignment demonstrates the use of Python file handling operations including reading, writing, and appending data to files. It simulates a simple sales data processing system and covers different file modes and operations.

## Tasks Covered
- Task 1: Write sales records to a file
- Task 2: Read file using read(), readline(), and readlines()
- Task 3: Append new sales data to the file
- Task 4: Generate summary report (total, max, min, average)
- Task 5: Create product info file using user input
- Task 6: Read file safely using existence check
- Task 7: Mini project - Export discounted prices to a file

## Concepts Used
- File modes: `r`, `w`, `a`
- File methods: `read()`, `readline()`, `readlines()`
- Writing and appending data
- Data processing from text files
- Safe file handling using `with open()`
- Basic use of `os.path.exists()`

## How to Run
1. Open the notebook: `file_handling.ipynb`
2. Run Task 1 first to generate `sales_data.txt`
3. Run the remaining tasks sequentially
4. Provide input when prompted (Task 5 & Task 7)

## Files Created
- `sales_data.txt` → stores sales values
- `products.txt` → stores product details
- `discount_report.txt` → stores discounted price report
